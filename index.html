DRR VIS (Vehicle Inspection System)

1. บทนำ

แอปพลิเคชัน DRR VIS เป็นระบบแสดงข้อมูลการชั่งน้ำหนักรถบรรทุกทั่วประเทศของกรมทางหลวงชนบท นโยบายความเป็นส่วนตัวนี้อธิบายวิธีการที่เราจัดการข้อมูลในการใช้งานแอปพลิเคชัน

2. ข้อมูลที่เราเก็บรวบรวม

2.1 ข้อมูลตำแหน่ง

  • ข้อมูลพิกัด GPS สำหรับแสดงตำแหน่งจุดชั่งน้ำหนัก (SPOT)
  • การเข้าถึงตำแหน่งที่ตั้งของอุปกรณ์เพื่อแสดงตำแหน่งปัจจุบันของผู้ใช้บนแผนที่

2.2 ข้อมูลการชั่งน้ำหนัก

  • ภาพถ่ายรถบรรทุกที่เข้าชั่งน้ำหนัก ณ จุดตรวจ (SPOT Check)
  • ข้อมูลน้ำหนักรถและรายละเอียดการชั่ง

2.3 ข้อมูลระบบพื้นฐาน

  • ข้อมูลอุปกรณ์: รุ่นอุปกรณ์และระบบปฏิบัติการเพื่อการทำงานที่เข้ากันได้
  • ข้อมูลการเชื่อมต่อพื้นฐานสำหรับการแสดงผลข้อมูล

3. การใช้ข้อมูล

เราใช้ข้อมูลเพื่อ:

  • แสดงตำแหน่งจุดชั่งน้ำหนักบนแผนที่
  • แสดงข้อมูลการชั่งน้ำหนักรถบรรทุกจากระบบฐานข้อมูลกลาง
  • ปรับปรุงประสิทธิภาพการทำงานของแอปพลิเคชัน

4. การแบ่งปันข้อมูล

ข้อมูลทั้งหมดที่แสดงในแอปพลิเคชันเป็นข้อมูลสาธารณะของกรมทางหลวงชนบท เราไม่มีการเก็บหรือแบ่งปันข้อมูลส่วนบุคคลของผู้ใช้งาน

5. ความปลอดภัยของข้อมูล

เรามีมาตรการรักษาความปลอดภัย:

  • การเข้ารหัสข้อมูลในการสื่อสารกับระบบฐานข้อมูล
  • การตรวจสอบความปลอดภัยของระบบอย่างสม่ำเสมอ
  • การจำกัดการเข้าถึงข้อมูลตามสิทธิ์ที่เหมาะสม

6. การเข้าถึงตำแหน่งที่ตั้ง

  • แอปพลิเคชันจะขออนุญาตเข้าถึงตำแหน่งที่ตั้งเพื่อแสดงตำแหน่งปัจจุบันของผู้ใช้บนแผนที่
  • ผู้ใช้สามารถอนุญาตหรือปฏิเสธการเข้าถึงตำแหน่งที่ตั้งได้ผ่านการตั้งค่าของอุปกรณ์
  • การปฏิเสธการเข้าถึงตำแหน่งที่ตั้งจะไม่ส่งผลต่อการแสดงข้อมูลจุดชั่งน้ำหนักและข้อมูลอื่นๆ ในระบบ

7. การเก็บรักษาข้อมูล

ข้อมูลที่แสดงในแอปพลิเคชันเป็นข้อมูลที่ได้จากระบบฐานข้อมูลกลางของกรมทางหลวงชนบท โดยมีระยะเวลาการเก็บรักษาดังนี้:

  • ข้อมูลการชั่งน้ำหนักและภาพถ่ายจะถูกเก็บไว้ในระบบเป็นระยะเวลา 5 ปี ตามระเบียบการจัดเก็บข้อมูลของกรมทางหลวงชนบท
  • ข้อมูลพิกัดตำแหน่งของจุดชั่งน้ำหนักจะถูกเก็บไว้ตลอดระยะเวลาที่จุดชั่งน้ำหนักยังคงให้บริการ
  • ข้อมูลตำแหน่งของผู้ใช้จะไม่ถูกบันทึกหรือจัดเก็บในระบบ

8. การใช้คุกกี้และเทคโนโลยีการติดตาม

แอปพลิเคชัน DRR VIS ไม่มีการใช้คุกกี้หรือเทคโนโลยีการติดตามใดๆ ในการเก็บข้อมูลผู้ใช้งาน

9. สิทธิ์ของผู้ใช้

ตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) ผู้ใช้มีสิทธิ์ดังต่อไปนี้:

  • สิทธิในการเข้าถึงข้อมูล: ผู้ใช้สามารถขอเข้าถึงข้อมูลที่เกี่ยวข้องกับตนเองได้
  • สิทธิในการควบคุมการใช้งานข้อมูล: ผู้ใช้สามารถควบคุมการเข้าถึงตำแหน่งที่ตั้งผ่านการตั้งค่าของอุปกรณ์
  • สิทธิในการร้องเรียน: ผู้ใช้มีสิทธิ์ร้องเรียนต่อหน่วยงานที่เกี่ยวข้องหากมีข้อสงสัยเกี่ยวกับการจัดการข้อมูล

10. การปฏิบัติตามกฎหมาย PDPA

กรมทางหลวงชนบทปฏิบัติตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) ในการจัดการข้อมูลทั้งหมดที่เกี่ยวข้องกับแอปพลิเคชัน DRR VIS โดยมีการกำหนดมาตรการที่เหมาะสมในการคุ้มครองข้อมูลส่วนบุคคลตามที่กฎหมายกำหนด

11. การเปลี่ยนแปลงนโยบาย

กรมทางหลวงชนบทอาจปรับปรุงนโยบายความเป็นส่วนตัวตามความเหมาะสม การเปลี่ยนแปลงจะมีผลเมื่อมีการอัปเดตแอปพลิเคชัน

1. Introduction

The DRR VIS application is a system displaying truck weighing data nationwide from the Department of Rural Roads. This privacy policy explains how we manage information when using the application.

2. Information We Collect

2.1 Location Data

  • GPS coordinates for displaying weighing station locations (SPOT)
  • Device location access for showing user's current position on the map

2.2 Weighing Data

  • Photographs of trucks taken during weighing at SPOT Check points
  • Vehicle weight data and weighing details

2.3 Basic System Data

  • Device information: device model and operating system for compatibility
  • Basic connection data for displaying information

3. Use of Information

We use the information to:

  • Display weighing station locations on the map
  • Show truck weighing data from the central database
  • Improve application performance

4. Information Sharing

All information displayed in the application is public data from the Department of Rural Roads. We do not collect or share personal user information.

5. Data Security

We implement security measures including:

  • Data encryption for database communications
  • Regular system security audits
  • Appropriate access control restrictions

6. Location Access

  • The application will request location access to show your current position on the map
  • Users can allow or deny location access through device settings
  • Denying location access will not affect the display of weighing station data and other system information

7. Data Retention

The information displayed in the application is obtained from the Department of Rural Roads' central database with the following retention periods:

  • Weighing data and photographs are stored in the system for 5 years according to the Department of Rural Roads' data retention regulations
  • Location coordinates of weighing stations are retained as long as the weighing stations remain operational
  • User location data is not recorded or stored in the system

8. Cookies and Tracking Technologies

The DRR VIS application does not use cookies or any tracking technologies to collect user data

9. User Rights

According to the Personal Data Protection Act B.E. 2562 (PDPA), users have the following rights:

  • Right to access: Users can request access to their relevant data
  • Right to control data usage: Users can control location access through device settings
  • Right to complain: Users have the right to file complaints with relevant authorities if they have concerns about data handling

10. PDPA Compliance

The Department of Rural Roads complies with the Personal Data Protection Act B.E. 2562 (PDPA) in managing all data related to the DRR VIS application, implementing appropriate measures to protect personal data as required by law

11. Policy Changes

The Department of Rural Roads may update this privacy policy as appropriate. Changes will take effect when the application is updated.

Effective Date: June 9, 2025
